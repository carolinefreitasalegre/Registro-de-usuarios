# Registro-de-usuarios

Este repositório contém um projeto de cadastro e registro de usuários, desenvolvido com objetivo de estudo e prática.  
O foco do projeto é reforçar conhecimentos sobre backend, persistência de dados, autenticação (se aplicável) e estruturação de aplicações com camadas.

## Objetivo do projeto

O sistema permite registrar usuários, armazenar seus dados e gerenciar operações básicas sobre essas informações.  
O projeto serve como ambiente de aprendizado para:

- Criação de APIs ou aplicações com backend  
- Persistência de dados em banco relacional  
- Organização de código em camadas (modelo, repositório, lógica, controle)  
- Compreensão de fluxo de cadastro e manipulação de dados  
- Boas práticas básicas de programação  

Este projeto não tem a intenção de ser usado em produção — serve apenas para estudo e fixação de conceitos.

## Tecnologias utilizadas

- .NET / C#  
- Entity Framework Core (ou outra ORM)  
- Banco de dados relacional (SQL Server, PostgreSQL, etc.)  
- API REST (ou outra abordagem conforme o projeto)
